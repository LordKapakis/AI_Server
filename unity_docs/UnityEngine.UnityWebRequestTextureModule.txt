Description


The UnityWebRequestTexture module provides the DownloadHandlerTexture class to use UnityWebRequest to download Textures.






Classes








Class


Description






DownloadHandlerTexture
A DownloadHandler subclass specialized for downloading images for use as Texture objects.


UnityWebRequestTexture
Helpers for downloading image files into Textures using UnityWebRequest.








Structs








Struct


Description






DownloadedTextureParams
Parameters for the texture to be created.








Enumerations








Enumeration


Description






DownloadedTextureFlags
Flags to be set in DownloadedTextureParams to indicate properties of the texture that will be created.
