Description


Describes the state of an accessibility node.






Properties








Property


Description






None
The accessibility node is in none of the other states.


Disabled
The accessibility node is currently in a disabled state and does not respond to user interaction.


Selected
The accessibility node is currently in a selected state (for example, a selected row in a table or a selected button within a segmented control).
